# UIデザイン仕様

## 全体
- ダークトーン背景、ゴールド/マゼンタのアクセント
- serif（Noto Serif JP）とゴシック（Zen Kaku Gothic New）を併用
- 余白を広く取り、アートブックのような雰囲気

## 作品追加ページ（upload.html）

### ステップUI
- 4ステップ表示（入力 → AI生成 → 確認・編集 → 完了）
- 現在ステップをゴールドで強調

### アップロードエリア
- 破線枠のドラッグ＆ドロップ
- プレビュー表示時は枠内に画像表示
- JPG/PNG、最大10MBの説明を明示

### 入力フォーム
- タイトル/補足メモ
- 任意入力であることを明示

### プレビュー
- 画像 + AI生成バッジ
- タイトルはインライン編集
- コメントはテキストエリアで編集可能

### 完了画面
- 成功アイコンと作品集への導線
- 追加ボタンで再入力へ戻す

## 作品集（index.html）
- 3Dカルーセル風の作品カード表示
- アクティブカードは拡大・前面表示
- 左右ボタン、ドット、スワイプで操作
- 「Works」を2秒以上長押しするとアップロード画面へ遷移（管理者用の隠し導線）
- 作品説明の下に「いいね」「コメント」のアイコンとカウントを表示
- コメントアイコン押下でコメント一覧を表示し、その下に入力フォームを配置

## 作品管理（admin.html）
- 作品一覧カードでタイトル/解説/画像を編集
- 画像差し替えはURL入力またはファイルアップロード
- 表示順を上下ボタンで変更
- 非表示/削除はソフト削除（データは保持）

## 資格セクション
- その他取得した資格のアイコン表示は削除（テキストのみ）

## アクセシビリティ
- 主要ボタンに明確なラベル
- エラーはテキストで明示
- キーボード左右キーで作品切り替え
