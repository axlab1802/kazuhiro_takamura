# 機能要件定義

## 目的
花の写真をアップロードするとAIがコメントを生成し、承認後に作品集へ追加できるLP機能を提供する。

## 対象ユーザー
- 一般ユーザー（写真アップロード）
- 管理者（AI生成結果の確認・承認）

## 機能要件

### 1. 作品アップロード
- JPG/PNG画像をドラッグ＆ドロップ/ファイル選択でアップロードできること
- 最大10MBまで受け付けること
- アップロード完了後、画像URLを取得できること

### 2. AI解析（コメント生成）
- 画像をAI（Gemini 3.0 Flash）に送信し、タイトルとコメントを生成できること
- ユーザーが指定した補足情報（タイトル/補足メモ）をプロンプトに反映できること
- 再生成ができること

### 3. 承認・作品集反映
- 生成されたタイトル/コメントを編集できること
- 承認操作で作品集に追加できること
- 作品集は新しい作品が先頭に表示されること

### 4. 作品集表示
- 作品一覧をAPI経由で取得できること
- データが存在しない場合は静的JSONにフォールバックできること

## 非機能要件
- 画像/コメント生成の処理は30秒以内に完了すること（Vercel Functionsの上限内）
- APIエラー時にユーザーへ分かりやすいメッセージを表示すること
- APIキー等の秘密情報は環境変数で管理すること
- 作品集表示は既存LPのデザインを崩さないこと
