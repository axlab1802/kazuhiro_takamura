# TAKAMURA KAZUHIRO | Floral Designer

髙村 和広 フラワーデザイナーのランディングページ

## 🌸 概要

回り道の先に、花が咲いた。  
15歳で社会に出て、バンド活動を経て、フラワーデザイナーとして新たな人生を歩む髙村和広の物語を伝えるLPサイトです。

## 🚀 技術スタック

- **Vite** - 高速な開発環境とホットリロード
- **HTML/CSS** - モダンなレスポンシブデザイン
- **JavaScript** - インタラクティブなアニメーション
- **Vercel** - 自動デプロイ対応

## 📦 セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動（ポート1105）
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## 🎨 特徴

- レスポンシブデザイン（PC/タブレット/スマホ対応）
- スクロールアニメーション
- モダンなタイポグラフィ
- グラデーション背景とカラーアクセント
- ギャラリー機能

## 📂 ディレクトリ構造

```
kazuhiro_Takamura/
├── index.html          # メインHTMLファイル
├── public/             # 静的ファイル
│   └── assets/         # 画像アセット
├── vite.config.js      # Vite設定
├── vercel.json         # Vercel設定
├── package.json        # 依存関係
└── .gitignore          # Git除外設定
```

## 🌐 デプロイ

Vercelで自動デプロイ設定済み。  
mainブランチへのプッシュで自動的にデプロイされます。

## 👨‍💻 開発情報

- 開発サーバー: `http://localhost:1105`
- ホットリロード: 有効
- ビルド出力: `dist/`
